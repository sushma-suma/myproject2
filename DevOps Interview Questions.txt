9. What is the diff b/w COPY and ADD in dockerfile? (100% surely asking in interviews)

  ans. Both COPY and ADD have the capability to copy files from the host machine into the customize image
  
  not container, but ADD can also download files from remote servers, COPy can't do that. [OR]

  ans. COPY simply the copy files which are present on the host machine into customized image we are creating
  
  where as ADD can do the same thing apart from that ADD also can download files from some remote servers into
  
  the customized images.
